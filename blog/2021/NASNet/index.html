2017л…„м—җ NASNetмқҙ лӮҳмҳЁ мқҙнӣ„ 2018л…„ л¶Җн„° AutoMLмқҳ мӢңлҢҖк°Җ м—ҙл ёмҠөлӢҲлӢӨ. MnasNet, MobileNet V3, EffientNetл“ұ м—¬лҹ¬ кІҪлҹүнҷ” л„ӨнҠёмӣҢнҒ¬л“ӨмқҖ NAS(Neural Architecture Search)мқ„ мӮ¬мҡ©н–Ҳкі , лӘЁл°”мқјмӘҪ л„ӨнҠёмӣҢнҒ¬л“ӨмқҖ л§Һмқҙ NASлҘј мӮ¬мҡ©н•ҳкі  мһҲмҠөлӢҲлӢӨ. мқҙлІҲм—җ мҶҢк°ңн•  л…јл¬ёмқҖ Neural Architecture Search with Reinforcement LearningлҘј мҶҢк°ңн•ҙл“ңлҰ¬кІ мҠөлӢҲлӢӨ.

Neural Architecture SearchмқҖ RNNмқ„ мӮ¬мҡ©н•ҳм—¬ model descriptionмқ„ мғқм„ұн•ҳкі  мғқм„ұлҗң л„ӨнҠёмӣҢнҒ¬лҘј н•ҷмҠөн•©лӢҲлӢӨ. к·ёлҰ¬кі  validation setмқ„ мқҙмҡ©н•ҳм—¬ accuracyлҘј кө¬н•ҳкі  мқҙлҘј rewardлЎң л§Ңл“ӯлӢҲлӢӨ. нӣ„м—җ мқҙ rewardлҘј к°–кі  policy gradientлҘј кө¬н•ҙ controllerлҘј м—…лҚ°мқҙнҠё мӢңнӮөлӢҲлӢӨ.

Generate Model Description

NASм—җм„ң controllerк°Җ model descriptionмқ„ л§Ңл“ӯлӢҲлӢӨ. model descriptionмқ„ к°–кі  modelмқ„ л§Ңл“  лӢӨмқҢ н•ҷмҠөмқ„ мӢңнӮӨкі  мҲҳл ҙмқҙ лҗҳл©ҙ validation setмңјлЎң accuracyлҘј мёЎм •н•ҳкІҢ лҗ©лӢҲлӢӨ. controller RNNмқҳ нҢҢлқјлҜён„° \(\theta_c\)лҠ” validation accuracyмқҳ нҸүк· мқ„ мқҙмҡ©н•ҙ мөңм Ғнҷ” мӢңнӮөлӢҲлӢӨ.

Training With REINFORCE

controllerлҘј н•ҷмҠөмӢңнӮӨл Өл©ҙ мҡ°лҰ¬лҠ” loss н•ЁмҲҳлҘј м •мқҳн•ҙм•јн•©лӢҲлӢӨ. controllerм—җм„ң л§Ңл“Өм–ҙм§„ л„ӨнҠёмӣҢнҒ¬лҘј child networkлқјкі  н•ҳкІ мҠөлӢҲлӢӨ. к·ёлҰ¬кі  child networkлҘј кө¬м„ұн•ҳкё° мң„н•ң actionл“Өмқ„ \(a_{1:T}\), н•ҷмҠөлҗң child networkмқҳ validation accuracyлҘј Rмқҙлқјкі  н•  л•Ң \(J(\theta_c)\)лҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

$$J(\theta_c)=E_{P(a_{1:T};\theta_c)}[R]$$

reward signalмқё RмқҖ лҜёл¶„ л¶Ҳк°ҖлҠҘн•ҳкё° л•Ңл¬ём—җ REINFORCEлқјлҠ” policyлЎң updateлҘј н•©лӢҲлӢӨ. мқҙлҠ” лӢӨмқҢкіј к°ҷмқҖ мӢқмңјлЎң л§Ңл“Ө мҲҳ мһҲкі 

$$\nabla_{\theta_c} J(\theta_c) = \sum_{t=1}^{T}E_{P(a_1:T; \theta_c)}[\nabla_{\theta_c} logP(a_t | a_{(t-1):1}; \theta_c) R]$$

л”°лқјм„ң мң„мқҳ мӢқмқҖ лӢӨмқҢкіј к°ҷмқҙ к·јмӮ¬лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

$$\frac{1}{m} \sum_{k=1}^{m} \sum_{t=1}^{T} \nabla_{\theta_c} logP(a_t | a_{(t-1):1}; {\theta_c}) R_k$$

mмқҖ н•ң л°°м№ҳм—җ controllerк°Җ л§Ңл“Ө child networkмқҳ к°ңмҲҳмқҙкі  kлІҲм§ё child networkмқҳ validation accuracyлҘј \(R_k\)мқҙлӢӨ. мң„мқҳ мӢқмқҖ л¶ҲнҺё м¶”м •м№ҳмқҙм§Җл§Ң л¶„мӮ°мқҙ нҒ¬кё° л•Ңл¬ём—җ л¶„мӮ°мқ„ мӨ„мқҙкё° мң„н•ҙм„ң лӢӨмқҢкіј к°ҷмқҖ мӢқмқ„ м”ҒлӢҲлӢӨ.

$$\frac{1}{m} \sum_{k=1}^{m} \sum_{t=1}^{T} \nabla_{\theta_c} logP(a_t | a_{(t-1):1}; {\theta_c}) (R_k-b)$$

bлҠ” мқҙм „ л„ӨнҠёмӣҢнҒ¬л“Өмқҳ validation accuracyмқҳ м§ҖмҲҳмқҙлҸҷнҸүк· мһ…лӢҲлӢӨ.

Skip Connection

GoogleNet, ResNetк°ҷмқҖ л„ӨнҠёмӣҢнҒ¬л“ӨмқҖ skip connectionмқ„ нҶөн•ҙ м„ұлҠҘмқ„ лҶ’мһ…лӢҲлӢӨ. мқҙмҷҖ к°ҷмқҙ NASNetм—җм„ңлҸ„ skip connectionмқ„ мғқм„ұн•ҳкё° мң„н•ҙ лӢӨмқҢкіј к°ҷмқҙ м„Өм •н–ҲмҠөлӢҲлӢӨ.

layer Nм—җм„ң мқҙм „мқҳ layerмңјлЎң л¶Җн„° skip connectionмқҙ мһҲлҠ”м§ҖлҘј кІ°м •н•ҳлҠ” N-1к°ңмқҳ content-based sigmoid(anchor point)лҘј м¶”к°Җн•©лӢҲлӢӨ.

P(Layer j is an input to layer i) = $$sigmoid(v^T tanh(W_{prev} * h_j + W_{curr} * h_i))$$

\(h_j\)лҠ” jлІҲм§ё layerмқҳ controllerмқҳ hiddenstateмқҙкі  к·ё к°’мқҖ 0 л¶Җн„° N-1к№Ңм§Җ к°Җм§Ҳ мҲҳ мһҲлӢӨ.

н•ҳм§Җл§Ң мқҙл ҮкІҢ м—°кІ°н•ҳлӢӨліҙл©ҙ кө¬мЎ°к°Җ л§қк°Җм§ҖлҠ” кІҪмҡ°к°Җ мһҲкё° л•Ңл¬ём—җ лӢӨмқҢкіј к°ҷмқҖ к·ңм№ҷмқ„ м •н•ңлӢӨ.

  1. input layerлЎң м“ҙ layerлҠ” лӢӨлҘё layerмқҳ input layerк°Җ лҗҳм§Җ м•ҠлҠ”лӢӨ.
  2. skip connectionмқҙ м•Ҳлҗҳм–ҙмһҲлҠ” layerлҘј лӢӨ final layerм—җ connectionмқ„ л§Ңл“ лӢӨ.
  3. л§Ңм•Ҫ skip connectionмӢң layerмқҳ sizeк°Җ лӢӨлҘҙл©ҙ zero paddingмқ„ н•ңлӢӨ.

Opinion

NASNetмқҳ м •нҷ•лҸ„ мһҗмІҙлҠ” лӢ№мӢң мҲҳлҸҷ м„Өкі„лҗң л„ӨнҠёмӣҢнҒ¬(ResNet, DenseNet л“ұ)мҷҖ л№„көҗн•ҙ м••лҸ„м Ғмқҙм§ҖлҠ” м•Ҡм§Җл§Ң, мӮ¬лһҢмқҳ м§ҒкҙҖ м—Ҷмқҙ кё°кі„к°Җ м„Өкі„н–ҲлӢӨлҠ” м җм—җм„ң нҒ° мқҳмқҳк°Җ мһҲмҠөлӢҲлӢӨ.

лӢӨл§Ң NASмқҳ мӢӨмҡ©м„ұм—җлҠ” н•ңкі„к°Җ мһҲм—ҲмҠөлӢҲлӢӨ. ControllerлҘј н•ҷмҠөмӢңнӮӨкё° мң„н•ҙ мҲҳл°ұ~мҲҳмІң к°ңмқҳ child networkлҘј н•ҷмҠөн•ҙм•ј н•ҳлҜҖлЎң, л…јл¬ём—җм„ңлҠ” 800лҢҖмқҳ GPUлЎң 28мқј лҸҷм•Ҳ мӢӨн—ҳн–ҲмҠөлӢҲлӢӨ. мқҙ л№„мҡ© л¬ём ңлҠ” мқҙнӣ„ DARTS(differentiable NAS), Once-for-All л“ұ нҡЁмңЁм Ғмқё NAS л°©лІ•лЎ мқҳ м—°кө¬ лҸҷкё°к°Җ лҗ©лӢҲлӢӨ.

к·ёлҰ¬кі  мӢӨн—ҳ кІ°кіјм—җм„ң filterл“Өмқҙ м§ҒмӮ¬к°Ғнҳ•мқҙ л§ҺлӢӨлҠ” кҙҖм°°мқҙ мһҲлҠ”лҚ°, мқҙлҠ” л¬ҙмһ‘мң„лЎң лҪ‘м•„лҸ„ м§ҒмӮ¬к°Ғнҳ•мқҙ лҗ  нҷ•лҘ мқҙ м •мӮ¬к°Ғнҳ•ліҙлӢӨ лҶ’кё° л•Ңл¬ёмқј мҲҳ мһҲмҠөлӢҲлӢӨ. NASк°Җ л°ңкІ¬н•ң нҢЁн„ҙмқҙ м •л§җ мқҳлҜёмһҲлҠ” кІғмқём§Җ, лӢЁмҲңн•ң нҷ•лҘ м Ғ кІ°кіјмқём§Җ кө¬л¶„н•ҳкё° м–ҙл Өмҡҙ м җмқҙ NAS м—°кө¬мқҳ н•ҙм„қ лӮңмқҙлҸ„лҘј лҶ’мқҙлҠ” мҡ”мҶҢмһ…лӢҲлӢӨ.
