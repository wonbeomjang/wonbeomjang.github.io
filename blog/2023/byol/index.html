Self-supervised learningВЮђ ваѕВЮ┤вИћВЮ┤ ВДђВаЋвљўВДђ ВЋіВЮђ вЇ░ВЮ┤ьё░ВЌљВёю featureвЊцВЮё ВХћВХюьЋўЖИ░ ВюёьЋ┤ ьЋЎВіхьЋўвіћ в░Ев▓ЋВЮ┤вІц. Ж│аВаёВаЂВЮИ в░Ев▓ЋВю╝вАювіћ AutoEncoder, GANВЮ┤ ВъѕВДђвДї contrastive learningВЮ┤ ВаюВІювљўЖ│а ьЂ░ Вё▒віЦьќЦВЃЂВЮё Ж░ђВаИВЎћвІц. ьЋўВДђвДї contrastive learningВЮђ negative labelВЮў ВѕўВЌљ ВЮўВА┤ВаЂВЮ┤Ж│а color distortionВЌљ в»╝Ж░љьЋўвІцвіћ вІеВаљВЮ┤ ВъѕвІц. ве╝Вађ contrastive learningВЌљ вїђьЋ┤ Вѓ┤ьј┤в│┤Ж│а ВЮ┤вЦ╝ BYOLВЮ┤ Вќ┤вќ╗Ж▓ї ьЋ┤Ж▓░ьЋўВўђвіћВДђ Вѓ┤ьј┤в│┤Въљ.

Contrastive Learning

Contrastive learningВЮђ self-supervised learningВЮў ьЋю ВбЁвЦўВЮ┤вІц. ВЮ┤в»ИВДђВЌљВёю data augmentationВЮё ьєхьЋ┤ image patchвЦ╝ ВХћВХюьЋўЖ▓ї вљўвіћвЇ░ Ж░ЎВЮђ ВЮ┤в»ИВДђВЌљВёю вѓўВўе image patchвіћ representation vector spaceВЌљВёю distanceЖ░ђ ВъЉЖ▓ї, вІцвЦИ ВЮ┤в»ИВДђВЌљВёю вѓўВўе image patchвіћ representation vector spaceВЌљВёю distanceЖ░ђ ьЂгЖ▓ї вДївЊавІц. ВЮ┤вЦ╝ьєхьЋ┤ ВЮ┤в»ИВДђВЮў featureвЦ╝ вѓўьЃђвѓ┤віћ representation vectorвЦ╝ вДївЊавІц.

Contrastive learningВЮў вІеВаљ

1. Data augmentationВЌљ ВЮўВА┤ВаЂ

contrastive learningВЮђ ьЋю ВЮ┤в»ИВДђ вѓ┤ВЌљВёю positive sampleВЮё ВХћВХюьЋювІц. вћ░вЮ╝Вёю positive sampleвЊцВЮђ в╣ёВіиьЋю color histogramВЮё Ж░ђВДђЖ▓ї вљювІц. ВЮ┤вЦ╝ ЖиИвїђвАю contrastive learningВЌљ ВѓгВџЕьЋа Ж▓йВџ░ вёцьіИВЏїьЂгвіћ вІцвЦИ ВюаВџЕьЋю ВаЋв│┤ вїђВІа color histogramвДїВю╝вАю positive sampleЖ│╝ negative sampleВЮё ЖхгвХёьЋўЖ▓ї вљўВќ┤ вфевЇИВЮў Вё▒віЦВЮ┤ вѓ«ВЋёВДђЖ▓ївљювІц.

ВюёВЮў ЖиИвд╝ВЮђ SimCLRВЮў data augmentationВю╝вАю data augmentationВЮў ВА░ьЋЕВЌљвЈё Вё▒віЦВЮ┤ вДјВЮ┤ вІгвЮ╝ВДёвІц.

2. Negative sampleВЮў ВѕўВЌљ ВЮўВА┤ВаЂ

\[\mathcal{L}_q=-log \frac{exp(q \cdot k_{+} / \tau)}{\sum_{i=0}^K exp(q \cdot k_i / \tau)}\]

Вюёвіћ contrastive learningВЮў loss functionВЮИ InfoNCEВЮ┤вІц. ВЮ┤ вЋї contrastive learningВЌљВёю positive sampleвДї ВѓгВџЕьЋўвЕ┤ вёцьіИВЏїьЂгвіћ ьЋГВЃЂ Ж░ЎВЮђ constant vectorвЦ╝ ВХюваЦьЋўВЌг model collapseЖ░ђ в░юВЃЮьЋювІц. вћ░вЮ╝Вёю ВЮ┤вЦ╝ в░ЕВДђьЋўЖИ░ ВюёьЋ┤ negative sampleЖ│╝ВЮў в╣ёЖхљвЈё ьЋўВЌг model collapseвЦ╝ в░ЕВДђьЋювІц.

$$\mathcal{L}_N^{opt}=-\underset{X}{\operatorname{\mathbb{E}}} log \left[ \frac{\frac{p(x_{t+k}|c_t)}{p(x_{t+k})}}{\frac{p(x_{t+k}|c_t)}{p(x_{t+k})} + \sum_{x_j \in X_{neg}} \frac{p(x_{j}|c_t)}{p(x_{j})}} \right]$$ $$= \underset{X}{\operatorname{\mathbb{E}}} log \left[ 1 + \frac{p(x_{t+k})}{p(x_{t+k}|c_t)} \underset{x_j \in X_{neg}}{\operatorname{\sum}} \frac{p(x_{j}|c_t)}{p(x_{j})} \right]$$ $$\approx \underset{X}{\operatorname{\mathbb{E}}} log \left[ 1 + \frac{p(x_{t+k})}{p(x_{t+k}|c_t)} (N-1) \underset{x_j}{\operatorname{\mathbb{E}}} \frac{p(x_{j}|c_t)}{p(x_{j})} \right]$$ $$= \underset{X}{\operatorname{\mathbb{E}}} log \left[ 1 + \frac{p(x_{t+k})}{p(x_{t+k}|c_t)} (N-1) \right]$$ $$\geq \underset{X}{\operatorname{\mathbb{E}}} log \left[ \frac{p(x_{t+k})}{p(x_{t+k}|c_t)} N \right]$$ $$= -I(x_{t+k}, c_t) + log(N)$$ $$I(x_{t+k}, c_t) \geq log(N) - \mathcal{L}_N^{opt}$$

ВЮ┤віћ loss functionВЌљВёювЈё в│╝ Вѕў ВъѕвІц. InfoNCEЖ░ђ optimal valueЖ░ђ вљўВЌѕВЮё вЋї ВаёЖ░юьЋ┤в│┤вЕ┤ вІцВЮїЖ│╝ Ж░ЎвІц. вДе ВЋёвъўВЮИ ВІЮВЮё ьЋ┤ВёЮьЋўВъљвЕ┤ infoNCEвіћ Mutual informationВЮў lower boundвЦ╝ ВаюьЋюьЋювІц. ВЮ┤ вЋї negative sampleВЮў Вѕў вўљьЋю lower boundвЦ╝ ВаюьЋюьЋ┤ negative sampleВЮў ВѕўЖ░ђ вДјВЮёВѕўвАЮ Вё▒віЦВЮ┤ ьќЦВЃЂвљўвіћ Ж▓ЃВЮё ВЋї Вѕў ВъѕвІц.


SimCLRВЮђ batchВЌљ positive sampleЖ│╝ negative sampleВЮё ьЋеЖ╗ў ЖхгВё▒ьЋўВЌг negative sampleВЮў ВѕўвЦ╝ ьЎЋв│┤ьЋювІц. ВЮ┤вЪгьЋю ьі╣Вё▒Вю╝вАюВЮИьЋ┤ batch sizeЖ░ђ ВцёВќ┤вЊцвЕ┤ Вё▒віЦВЮ┤ ЖИЅЖ▓ЕьЋўЖ▓ї ьЋўвЮйьЋўвіћ Ж▓ЃВЮё в│╝ Вѕў ВъѕвІц.

вўљьЋю MoCoвіћ momentum encoderвЦ╝ ВѓгВџЕьЋўВЌг negative sample queueвЦ╝ вДївЊцВќ┤ negative sampleвЦ╝ ьЎЋв│┤ьЋювІц. ВЮ┤вЪгьЋю ьі╣Вё▒Вю╝вАю batch sizeЖ░ђ ВъЉВЋёвЈё вДјВЮђ negative sampleВЮё ьЎЋв│┤ьЋа Вѕў ВъѕвІц.

BYOL

ВЮ┤ вфевЊа Ж│╝ВаЋВЮё model collapseвЦ╝ в░ЕВДђьЋўЖИ░ ВюёьЋ┤ ьЋўвіћ Ж▓ЃВЮ┤вІц. BYOLВЮў ВађВъљвіћ ВЮ┤вЦ╝ ьЋ┤Ж▓░ьЋўЖИ░ВюёьЋ┤ ЖиИВађ encoderвЦ╝ random initializationьЋўВЌг ВІцьЌўВЮё ВДёьќЅьќѕвІц.


EncoderвЦ╝ random initializationьЋю ьЏё parameterвЦ╝ Ж│аВаЋьЋювІц. ЖиИвдгЖ│а linear layerвЦ╝ ьЋўвѓў вХЎВЌг ImageNetВЌљ ьЋЎВіхВЮё ВІюьѓг Ж▓йВџ░ ВаЋьЎЋвЈёЖ░ђ 1.8%вАю вѓ«ВЋўвІц. ьЋўВДђвДї random initialization encoderВЮў representation vectorвЦ╝ labelвАю вДївЊцВќ┤ encoderвЦ╝ ьЋЎВіхВІюьѓе ьЏё ВюёВЎђЖ░ЎВЮђ ьЋЎВіхВЮё ьЋўвЕ┤ ВаЋьЎЋвЈёЖ░ђ 18.8%вАю ВЃЂвІ╣ьъѕ ВЃЂВі╣ьќѕвІц. ВађВъљвіћ ВЮ┤ВЌљ ВЋёВЮ┤вћћВќ┤вЦ╝ Вќ╗Вќ┤ ВІцьЌўВЮё ВДёьќЅьЋўВўђвІц.

ьЋЎВіхЖ│╝ВаЋ

  1. ЖхгВА░Ж░ђ вЈЎВЮ╝ьЋю online networkВЎђ target networkвЦ╝ вДївЊавІц.
  2. Target networkВЮў ВЌГьЋаВЮђ online networkЖ░ђ ВўѕВИАьЋа representation vector \(f_\mathcal{E}\)вЦ╝ вДївЊавІц.
  3. Online networkВЮў ВЌГьЋаВЮђ \(f_\mathcal{E}\)вЦ╝ ВўѕВИАьЋўвіћ Ж▓ЃВЮ┤вІц.
  4. Representation vectorвЦ╝ ЖиИвїђвАю ВѓгВџЕьЋўВДђ ВЋіЖ│а projection layer \(g\)вЦ╝ ВѓгВџЕьЋўВЌг projection vector \(z\)вАю в│ђьЎўьЋювІц. (SimCLR В░ИЖ│а)
  5. \(z'_\mathcal{E}\)ВЮё ВўѕВИАьЋўЖИ░ВюёьЋ┤ Online networkВЌљ prediction layer \(q_{\theta}\)вЦ╝ ВХћЖ░ђьЋўВЌг asymmetricьЋю ЖхгВА░вЦ╝ вДївЊцВЌѕвІц.
  6. Data augmentation pool \(\mathcal{T}\) ВЮё вДївЊцЖ│а \(t\), \(t'\)вЦ╝ вйЉВЋё Ж░ЂЖ░ЂВЮў вёцьіИВЏїьЂгВЌљ ьєхЖ│╝ВІюьѓевІц.
$$\mathcal{L}_{\theta, \mathcal{E}} \triangleq \| \overline{q_{\theta}(z_{\theta})} - \overline{z'_\mathcal{E}} \|^2_2 = 2 - 2 \cdot \frac{\langle q_{\theta}(z_{\theta}), z'_\mathcal{E} \rangle}{\| q_{\theta}(z_{\theta}) \|_2 \cdot \| z'_\mathcal{E} \|_2}$$ $$\mathcal{L}^{BYOL}_{\theta, \mathcal{E}} = \mathcal{L}_{\theta, \mathcal{E}} + \tilde{\mathcal{L}}_{\theta, \mathcal{E}}$$ $$\theta \leftarrow optimizer(\theta, \nabla_{\theta} \mathcal{L}^{BYOL}_{\theta, \mathcal{E}}, \eta)$$ $$\mathcal{E} \leftarrow \tau \mathcal{E} + (1 - \tau) \theta$$
  1. online networkВЮў output \(q_{\theta}(z_{\theta})\)ВЎђ \(z'_\mathcal{E}\)вЦ╝ L2 normalizeьЋўВЌг \(\overline{q_{\theta}(z_{\theta})}\)ВЎђ \(\overline{z'_\mathcal{E}}\)вЦ╝ вДївЊавІц.
  2. \(\overline{q_{\theta}(z_{\theta})}\)ВЎђ \(\overline{z'_\mathcal{E}}\)ВЮў L2 lossвЦ╝ ЖхгьЋювІц.
  3. SymmetricьЋю ЖхгВА░вЦ╝ вДївЊцЖИ░ ВюёьЋ┤ \(t\), \(t'\)вЦ╝ вњцВДЉВќ┤ вІцВІю lossвЦ╝ ЖхгьЋювІц.
  4. Online networkВЮў parameterвіћ backpropagationВЮё ьєхьЋ┤ updateьЋювІц.
  5. Target networkВЮў parameterвіћ online networkВЮў parameterВЮў exponential moving average(EMA)ВЮё ВѓгВџЕьЋўВЌг updateьЋювІц.

Why it works?

вІеВѕюьъѕ в│┤ЖИ░вДїьЋ┤вЈё ЖхгВА░ВЎђ ьЋЎВіхв░Ев▓ЋВЮ┤ ВЮ┤ВЃЂьЋўЖ│а ВЎю ВъЉвЈЎьЋўвіћВДђ вфевЦ┤Ж▓авІц. ВађВъљвіћ РђюBYOL ВЮў prediction layerЖ░ђ optimal ВЮИ Ж▓йВџ░ВЌљвіћ undesirable equilibria віћ unstable ьЋўвІц.РђЮвЮ╝Ж│а ьЋювІц. ВдЅ, model collapseЖ░ђ в░юВЃЮьЋўЖИ░ Вќ┤вахвІцвіћ Ж▓ЃВЮ┤Ж│а вїђьЋю ВЮ┤ВюавАю prediction layerВЎђ EMAвЦ╝ вйЉВЋўвІц.

Prediction layer

вДїВЋй prediction layerЖ░ђ optimal solutionВЮ┤ \(p^{\star}\)вЮ╝Ж│а ьЋўВъљ.

$$p^{\star} \triangleq \underset{q}{\operatorname{argmin}} \mathbb{E} \left [\| q_{\theta}(z_{\theta}) - z'_\mathcal{E} \|^2_2 \right ], p^{\star}(z_{\theta})=\mathbb{E}\left [z'_\mathcal{E} | z_{\theta}\right ]$$

вІцВЮї parameter updateвіћ вІцВЮїЖ│╝ Ж░ЎвІц.

$$ \nabla_{\theta} \mathbb{E} \left [\| q^{\star}(z_{\theta}) - z'_{\mathcal{E}} \|^2_2 \right ] = \nabla_{\theta} \mathbb{E} \left [\| \mathbb{E}\left [z'_\mathcal{E} | z_{\theta}\right ] - z'_{\mathcal{E}} \|^2_2 \right ] = \nabla_{\theta} \mathbb{E} \left [\sum_i Var(z'_{\mathcal{E},i}|z_{\theta}) \right ] $$

вДїВЋй online networkЖ░ђ representationВЮё constantЖ░њВЮё ВХюваЦьЋювІцвЕ┤ вІцВЮїЖ│╝ Ж░ЎВЮђ ВІЮВЮ┤ Вё▒вдйьЋювІц.

$$Var(z'_{\mathcal{E}}|z_{\theta}) \leq Var(z'_{\mathcal{E}}|c)$$

ВЮ┤ вЋї constant representation vectorв│┤вІц ВъЉВЮђ representation vectorЖ░ђ ВА┤ВъгьЋўв»ђвАю вІцВЮї parameter updateЖ░ђ ВЮ╝Вќ┤вѓювІц.

EMA

Target networkвЦ╝ updateьЋа вЋї EMAвЦ╝ ВѓгВџЕьЋўВДђ ВЋіЖ│а gradient descentвАю updateьЋўвЕ┤ model collapseЖ░ђ в░юВЃЮьЋювІц.

$$Var(c|z_{\theta}) \leq Var(z'_{\mathcal{E},i}|z_{\theta})$$

ВюёВЮў ВІЮЖ│╝ Ж░ЎВЮђВЮ┤ВюавАю model collapseЖ░ђ в░юВЃЮьЋўвЕ┤ parameter updateЖ░ђ ВЮ╝Вќ┤вѓўВДђ ВЋівіћвІц.

Experimental Result

BackboneВю╝вАю ResNet50ВЮё ВѓгВџЕьЋўВўђвІц. ImageNetВЌљВёю linear evaluationВЌљВёю Вё▒віЦВЮ┤ ВбІВЋўЖ│а вІцвЦИ ResNet backboneВЌљВёювЈё Вё▒віЦВЮ┤ ВбІВЋўвІц.

ВаёВ▓┤ datasetВЮў ВЮ╝вХђвДї ВѓгВџЕьЋўвіћ semi-supervised learningВЌљВёювЈё Ж░ЎВЮђ Ж▓░Ж│╝вЦ╝ в│┤ВўђвІц

ImageNetВЌљвДї ьі╣ьЎћвљўВќ┤ВъѕвіћВДђ ВЋїЖИ░ВюёьЋ┤ вІцвЦИ classification taskВЌљ ВаЂВџЕВЮё ьќѕВЮё вЋївЈё вїђвХђвХёВЮў Ж▓йВџ░ВЌљ Вё▒віЦВЮ┤ ВбІВЋўвІц.

Ablation study

Batch size, data augmentationВЮў ВўЂьќЦВЮё в░ЏВю╝вѓў ВЮ┤віћ SimCLRв│┤вІц вЇћ ВаЂВЮђ Ж▓ЃВю╝вАю вѓўьЃђвѓювІц. BYOLВЮђ positive sampleвДї ВѓгВџЕьЋўЖИ░ вЋївгИВЌљ batch sizeВЌљ вЉћЖ░љьЋўвІцвіћ Ж▓ЃВЮ┤вЮ╝ ВХћВаЋьЋювІц.
