MobileOne: An Improved One millisecond Mobile Backbone

Introduction

MobileNet, ShuffleNetл“ұ кІҪлҹүнҷ” л„ӨнҠёмӣҢнҒ¬л“ӨмқҖ parameterмқҳ мҲҳмҷҖ FLOPSлҘј кё°мӨҖмңјлЎң лӘЁлҚёмқ„ кІҪлҹүнҷ”н•ҳкі  мһҲлӢӨ. MobileNetV3, MNasNet л“ұ л§Ңмқҙ мӢӨм ң кё°кё°м—җм„ң latencyлҘј мёЎм •н•ҳм—¬ л°ҳмҳҒн•ҳкі  мһҲлӢӨ. м ҖмһҗлҠ” parameter мҲҳмҷҖ FLOPSк°Җ мӨ„м–ҙл“ӨмҲҳлЎқ latencyк°Җ мӨ„м–ҙл“ңлҠ” кҙҖкі„к°Җ н•ӯмғҒ мқјм№ҳн•ҳм§Җ м•ҠлҠ”лӢӨлҠ” кІғмқ„ л°ңкІ¬н•ҳм—¬ мӢӨм ңлЎң лӘЁлҚёмқҳ м–ҙл–Ө л¶Җл¶„мқҙ лҶ’мқҖ ліөмһЎлҸ„лҘј к°–лҠ”м§Җ нҸүк°Җн•ҳм—¬ лӘЁлҚёмқ„ м ңмһ‘н–ҲлӢӨ.

Method

Metric Correlation

лӘЁлҚёмқҳ costлҘј л№„көҗн•  л•Ң parameterмқҳ мҲҳмҷҖ FLOPSлҘј кё°мӨҖмңјлЎң л№„көҗн•ҳкіӨн•ңлӢӨ. н•ҳм§Җл§Ң мӢӨм ң лӘЁл°”мқјнҷҳкІҪм—җм„ң мқјм№ҳк°Җ м•Ҳлҗ  кІҪмҡ°кі  мһҲмңјлҜҖлЎң iPhone 12лҘј мӮ¬мҡ©н•ҳм—¬ мғҒкҙҖкҙҖкі„лҘј мёЎм •н–ҲлӢӨ.

к·ё кІ°кіј лӘЁл°”мқј нҷҳкІҪм—җм„ңлҠ” FLOPSк°Җ мӨ‘к°„м •лҸ„мқҳ мғҒкҙҖкҙҖкі„лҘј лӮҳнғҖлғҲкі  нҢҢлқјлҜён„°мқҳ мҲҳлҠ” лӮ®мқҖ мғҒкҙҖкҙҖкі„лҘј лӮҳнғҖлғҲлӢӨ. л°ҳл©ҙм—җ CPU нҷҳкІҪм—җм„ңлҠ” мғҒкҙҖкҙҖкі„к°Җ м—Ҷм–ҙ ліҙмқёлӢӨ.

Key Bottlenecks

Activation Function

Activation functionл§ҲлӢӨ costк°Җ лӢӨлҘҙлӢӨ. л”°лқјм„ң мқҙлҘј л№„көҗн•ҳкё° мң„н•ҙ 30 layerмқҳ лӘЁлҚёмқ„ м„ м–ён•ҳмҳҖмңјл©° activation layerл§Ң л°”кҫёл©ҙм„ң мҶҚлҸ„мёЎм •мқ„ н–ҲлӢӨ.

Dynamic Shift-Maxл“ұ м—¬лҹ¬ к°•л Ҙн•ң activation functionмқҙ мһҲмңјлӮҳ мқҙлҠ” latencyк°Җ м»Өм„ң RELUлҘј мӮ¬мҡ©н•ҳкё°лЎң н–ҲлӢӨ.

Architectural Blocks

Runtime performanceм—җ мҳҒн–Ҙмқ„ мЈјлҠ” мӣҗмқёмқҖ нҒ¬кІҢ memory access costкіј degree of parallelismмқҙ мһҲлӢӨ. мҳҲлҘј л“Өл©ҙ skip connectionмқҖ мқҙм „ feature mapм—җ лҢҖн•ң м •ліҙлҘј м ҖмһҘн•ҳкі  мқҪм–ҙмҷҖм•јн•ҳкё° л•Ңл¬ём—җ memory access costк°Җ лҠҳм–ҙлӮҳкі  SE-blockм—җм„ң мӮ¬мҡ©н•ҳлҠ” global pooling operationмқҖ лҸҷкё°нҷ”лҘј н•ҙм•јн•ҳкё° л•Ңл¬ём—җ degree of parallelismмқҙ мӨ„м–ҙл“ лӢӨ. л”°лқјм„ң MobileOneм—җм„ңлҠ” skip connectionмқ„ м ңкұ°н•ҳкі  SE-blockмқҳ мҲҳлҠ” м ҒлӢ№нһҲ мЎ°м Ҳн•ҳмҳҖлӢӨ.

MobileOne Architecture

MobileOne Block

кё°ліём Ғмқё blockмқҖ depthwise, pointwise layerлЎң factorizationн•ҳмҳҖлӢӨ. Basic blockмқҖ MobileNet-V1м—җм„ң мӮ¬мҡ©н•ҳлҠ” 3x3 depthwise convolutionкіј 1x1 pointwise convolutionмқ„ мӮ¬мҡ©н•ңлӢӨ. к·ёлҰ¬кі  Rep-VGGм—җм„ң мӮ¬мҡ©н•ң re-parameterizable skip connectionмқ„ мӮ¬мҡ©н•ңлӢӨ. мқҙ л•Ң trivial over-parameterization factor kлҠ” 1~5мқҳ к°’мқ„ мӮ¬мҡ©н•ңлӢӨ. Memory access costлҘј мӨ„мқҙкё° мң„н•ҙ skip connectionмқҖ inference timeм—җ м ңкұ°н–ҲлӢӨ.

Convolutionм—җ лҢҖн•ҙм„ңлҠ” BatchNormмқ„ Conv layerм—җ foldingн•ҳмҳҖлӢӨ. Kernel size \(K\), Input channel dimension \(C_{in}\), output channel dimension \(C_{out}\)м—җ лҢҖн•ҙм„ң weight matixлҠ” \(W^\prime \in \mathbb{R}^{C_{out} \times C_{in} \times K \times K}\), biasлҠ” \(b^\prime \in \mathbb{R}^D\)лЎң н‘ңмӢңн•  мҲҳ мһҲлӢӨ. лҳҗн•ң BatchNormмқҖ accumulated mean \(\mu\), accumulated standard deviation \(\sigma\), scale \(\gamma\), bias \(\beta\)лЎң кө¬м„ұлҗҳм–ҙ мһҲлӢӨ. ConvмҷҖ BNмқҖ лӘЁл‘җ linear operationмқҙлҜҖлЎң мқҙлҘј лӢӨмқҢкіј к°ҷмқҙ н•©м№  мҲҳ мһҲлӢӨ.

  • Weight
\[\hat{W}=W^\prime * \frac{\gamma}{\sigma}\]
  • Bias
\[\hat{b}=(b^\prime - \mu) * \frac{\gamma}{\sigma}+\beta\]

BNм—җ лҢҖн•ң skip connectionмқҖ 1x1 convolutioм—җ K-1 zero paddingмңјлЎң folding н•  мҲҳ мһҲлӢӨ. мң„ кіјм •мқ„ нҶөн•ҙ м–»м–ҙм§„ foldingмқҖ inference timeм—җ мӮ¬мҡ©лҗңлӢӨ.

Model Scaling

лӘЁлҚёмқҳ мҶҚлҸ„лҘј мң„н•ҙ blockмқҳ мҲҳлҠ” resolutionл§ҲлӢӨ лӢӨлҘҙкІҢм„Өм •н–ҲлӢӨ. мқҙлҠ” resolutionмқҙ лҶ’мқҖ мғҒлӢЁ layerм—җм„ңлҠ” costк°Җ лҶ’кё° л•Ңл¬ём—җ blockмқҳ мҲҳлҘј мӨ„мқҙкі  н•ҳлӢЁ layerлҠ” channel мҲҳк°Җ л§Һкё° л•Ңл¬ём—җ blockлҘј мӨ„мқҙкІҢ лҗң кІғмқҙлӢӨ.

Training

Overfittingмқ„ л°©м§Җн•ҳкё° мң„н•ҙ small modelм—җ лҢҖн•ҙм„ңлҠ” regularizationмқ„ м ҒкІҢ н•ҙм•јн•ңлӢӨ. лҳҗн•ң cosine learning rateлҘј decayмҷҖ н•Ёк»ҳ regularizationлҸ„ deacyлҘј н•ҙмӨҖлӢӨ.

Result

нҢҢлқјлҜён„°мқҳ мҲҳк°Җ л№„мҠ·н•ң лӘЁлҚёмӨ‘м—җм„ңлҠ” м„ұлҠҘмқҙ м ңмқј лҶ’кІҢ лӮҳнғҖлӮңлӢӨ.

ReparameterizationлҸ„ м„ұлҠҘм—җ мўӢмқҖ мҳҒн–Ҙмқ„ лӮҙлҠ”лҚ° лӘЁлҚёмқҳ нҒ¬кё°к°Җ нҒҙмҲҳлЎқ м„ұлҠҘм°Ёмқҙк°Җ мўҒм•„м§„лӢӨ.

лӢӨлҘё лӘЁл°”мқј л„ӨнҠёмӣҢнҒ¬лҘј CPU, GPU, Mobileм—җм„ң лҸҢл ёмқ„ л•Ң м„ұлҠҘкіј мҶҚлҸ„к°Җ мўӢм•ҳкі 

knowledge distillationмқ„ н–Ҳмқ„ л•ҢлҸ„ м„ұлҠҘмқҙ м ңмқј мўӢм•ҳлӢӨ.
