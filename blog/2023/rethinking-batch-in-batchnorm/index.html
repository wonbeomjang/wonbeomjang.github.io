Rethinking вҖңBatchвҖқ in BatchNorm

Created on March 14, 2023

2023 В  В· В  paper В  batch-normalization В  training В  В· В  paper

Intorduction

BatchNormмқҖ layerмқҳ мӨ‘к°„м—җ л“Өм–ҙк°Җл©° н•ҷмҠөмқ„ м•Ҳм •нҷ”н•ҳм—¬ н•ҷмҠөмҶҚлҸ„лҘј мғҒмҠ№мӢңнӮӨкі  мҳӨлІ„н”јнҢ…мқ„ л°©м§Җн•ңлӢӨ. н•ҳм§Җл§Ң train test domainмқҙ л§Һмқҙ лӢӨлҘҙкұ°лӮҳ batch sizeк°Җ нҳ„м Җн•ҳкІҢ мһ‘лӢӨл©ҙ batch normмқҖ л¬ём ңлҘј мқјмңјнӮӨкё°лҸ„ н•ңлӢӨ. л”°лқјм„ң мқҙ л…јл¬ём—җм„ңлҠ” BatchNormмқҳ н•Ём •мқ„ м •лҰ¬н•ҳкі  к¶ҢмһҘмӮ¬н•ӯмқ„ м ңмӢңн•ңлӢӨ.

BatchNorm

BatchNormмқҖ лӢӨмқҢкіј к°ҷмқҖ мӢқмңјлЎң м •мқҳлҗңлӢӨ.

\[y=\frac{x-\mu}{\sqrt{\sigma^2+\epsilon}}\]

ліҙнҶө нҸүк· кіј л¶„мӮ°мқҖ н•ҷмҠөмӢң mini-batchлҘј нҶөн•ҳм—¬ лӘЁнҸүк· кіј лӘЁл¶„мӮ°мқ„ м¶”м •н•ңлӢӨ.

\[\mu_{\mathcal{B}}=mean(X,axis=[N,H,W])\] \[\sigma_{\mathcal{B}}^2=var(X,axis=[N,H,W])\]

кё°мЎҙмқҳ BatchNormмқҖ нқ”нһҲ лӘЁнҸүк· кіј лӘЁл¶„мӮ°мқ„ мҳҲмёЎн•ҳкё° мң„н•ҙ EMA(Exponential Moving Average)лҘј мӮ¬мҡ©н•ңлӢӨ.

\[\mu_{EMA} \leftarrow \lambda \mu_{EMA} + (1-\lambda) \mu_{\mathcal{B}}\] \[\sigma_{EMA}^2 \leftarrow \lambda \sigma_{EMA}^2 + (1-\lambda) \sigma_{\mathcal{B}}^2\]

н•ҳм§Җл§Ң EMAлҠ” мқҙм „м—җ мӮ¬мҡ©н•ң к°’мқ„ лҢҖл¶Җл¶„ к°Җм ёк°Җ мӢӨм ң нҸүк· , л¶„мӮ°мқ„ лҠҰкІҢ л°ҳмҳҒн•ңлӢӨ. (\(\lambda\)лҠ” ліҙнҶө 0.9лЎң м„Өм •н•ңлӢӨ.)

PresizeBN

л”°лқјм„ң м ҖмһҗлҠ” PreciseBNмқ„ м ңм•Ҳн–ҲлӢӨ. PresizeBNмқҖ mini-batchл§ҲлӢӨ batch parameterлҘј updateн•ҳм§Җ м•Ҡкі  н•ң epochмқҙ лҒқлӮҳл©ҙ modelмқ„ freezeн•ң нӣ„ batch statisticsлҘј aggregationмқ„ н•ҳм—¬ updateн•ңлӢӨ.

\[\mu_{pop}=E[\mu_{\mathcal{B}}], \sigma_{pop}^2=E[\mu_{\mathcal{B}}^2+\sigma_M^2]-E[\mu_{\mathcal{B}}^2]\]

EMA vs PreciseBN

Large Batch Size

PreciseBNмқҖ EMAліҙлӢӨ м•Ҳм •м ҒмқҙлӢӨ. м ҖмһҗлҠ” лЁјм Җ batch sizeк°Җ л§Өмҡ° нҒҙ кІҪмҡ°м—җ лҢҖн•ҳм—¬ мӢӨн—ҳн–ҲлӢӨ. EMAм—җм„ң batch sizeк°Җ м»Өм§Җл©ҙ м ҲлҢҖм Ғмқё updateмҲҳк°Җ м Ғм–ҙм ё validation errorмқҳ л¶„мӮ°мқҙ м»Өм§„лӢӨ. н•ҳм§Җл§Ң PreciseBNмқҖ н•ң epochмқҙнӣ„м—җ updateн•ҳкё° л•Ңл¬ём—җ л¶„мӮ°мқҙ м»Өм§ҖлҠ” мқјмқҙ м—Ҷм—Ҳкі  мӢӨн—ҳм ҒмңјлЎң 1k~10kмқҳ sampleмқ„ aggregationмқ„ н•ҳл©ҙ лӘЁмҲҳлҘј м¶”м •н•ҳкё° м¶©л¶„н–ҲлӢӨ.

Small Batch Size

EMAлҠ” mini-batchл§Ң ліј мҲҳ мһҲкё°л–„л¬ём—җ batch sizeк°Җ мһ‘мқ„мҲҳлЎқ м„ұлҠҘн•ҳлқҪмқҙ м»Ө PresiceBNмқҳ м„ұлҠҘмқҙ лҶ’м•ҳлӢӨ. н•ҳ м§Җл§Ң мӢӨн—ҳм ҒмңјлЎңлҠ” EMAлҠ” лӘЁлҚёмқҙ м¶©л¶„нһҲ н•ҷмҠөн•ҳм—¬ мҲҳл ҙн–Ҳмқ„ л•ҢмҷҖ Batch sizeк°Җ м¶©л¶„нһҲ нҒ° кІҪмҡ° Precise BNкіј м„ұлҠҘм°Ёмқҙк°Җ лі„лЎң м—Ҷм—ҲлӢӨ.

EMAлҠ” Batchsizeк°Җ мһ‘м•„м§ҲмҲҳлЎқ train-test inconsisitencyк°Җ м»Өм§„лӢӨ. л”°лқјм„ң inferenceм—җ mini-batch statisticмқ„ мқҙмҡ©н•ҳл©ҙ м„ұлҠҘн•ҳлқҪмқҙ мӨ„м–ҙл“ лӢӨ.

FrozenBN

Finetuningн•  л•Ң batch normмқ„ freezeн•ңлӢӨ. н•ҳм§Җл§Ң мқјл°ҳм Ғмқё trainingм—җм„ңлҸ„ нҡЁкіјлҘј ліҙлҠ” кІғмқ„ м°ҫм•„лғҲлӢӨ. мқҙ л•Ң н•ҷмҠө epochм—җм„ң мӨ‘к°„мҜӨл¶Җн„° batchnormмқ„ freezeн•ҳл©ҙ лҗңлӢӨ. ImageNetмӢӨн—ҳм—җм„ңлҠ” л§Ҳм§Җл§ү 20 epochл•Ң frozneBNмқ„ мӮ¬мҡ©н•ҳмҡ” train-test-inconsistencyлҘј к°ңм„ н–ҲлӢӨ.

Adaptive BatchNorm

Train testм—җм„ң нҒ° domain inconsistencyк°Җ мЎҙмһ¬н•ңлӢӨ. л”°лқјм„ң Test setм—җм„ң batch normмқҳ polution statisticsлҘј н•ҷмҠө нӣ„ нҸүк°Җн–Ҳмқ„ л•Ң м •нҷ•лҸ„к°Җ мғҒмҠ№н–ҲлӢӨ. мқҙ л•Ң trainкіј evalutationмқҳ augmentation л°©лІ•мқҙ лҸҷмқјн•ҙм•јн•ңлӢӨ.

BatchNorm in multi-domain training

multi-domainм—җм„ң лӢӨмқҢмқҳ мӢқмқҙ м„ұлҰҪн•ңлӢӨ.

\[f([X_1, X_2,...,X_n]) \neq [f(X_1, f(X_2), ..., f(X_n)]\]

мҰү, domain лі„лЎң normalizeн•ҳлҠ” кІғкіј domainмқ„ н•©міҗ normalizeн•ҳлҠ” кІғкіј лӢӨлҘҙлӢӨ. мқҙкІғмқҙ л¬ём ңлҘј мқјмңјнӮЁлӢӨ.

м ҖмһҗлҠ” мқҙлҘј retinanetм—җ мӢӨн—ҳмқ„ н–ҲлӢӨ. RetinanetмқҖ sizeк°Җ лӢӨлҘё featuremapмқ„ кіөмң лҗң headлЎң detectionмқ„ м§„н–үн•ңлӢӨ. н•ҳм§Җл§Ң мқҳ к°Ғк°Ғмқҳ feature mapмқҖ лӢӨлҘё domainмқ„ ліј мҲҳ мһҲлӢӨ. л”°лқјм„ң к°Ғк°Ғмқҳ featuremapмқ„ normaizationн•ҳлҠ” кІғмқҙ м•„лӢҢ н•©міҗм„ң normalizationмқ„ н•ҳл©ҙ м„ұлҠҘмқҙ лҶ’м•„м§„лӢӨ. мқҙ лҢҖ traningкіј population statisticsкіј affine parameterмқҳ нҷҳкІҪмқ„ мқјм№ҳн•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•ҳлӢӨ.

Ghost BatchNorm, LayerNorm

Trainingн•ҳлӢӨліҙл©ҙ mini-batchм•Ҳм—җ к°ҷмқҖ classмқҳ imageк°Җ л“Өм–ҙк°ҖлҠ” кІҪмҡ°к°Җ мһҲлӢӨ. мқҙлҹҙл•Ң лӘЁлҚёмқҖ mini-bathм—җм„ң classмқҳ hintлҘј м–»мқ„ мҲҳ мһҲкё° л•Ңл¬ём—җ mini-batchк°„ dependencyк°Җ мЎҙмһ¬н•  мҲҳ мһҲкі  мқҙлҠ” biasлҘј мң л°ңн•ңлӢӨ. л”°лқјм„ң minibatchм—җм„ң м„ңлЎң лӢӨлҘё class лҒјлҰ¬ normalizaionмқ„ м§„н–үн•ҳл©ҙ м„ұлҠҘмқҙ мҳ¬лқјк°„лӢӨ.
