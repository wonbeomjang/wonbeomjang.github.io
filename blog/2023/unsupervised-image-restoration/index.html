Invariant Representation for Unsupervised Image Restoration

Introduction

Image restorationмқҖ лҢҖл¶Җл¶„ pairн•ң datasetмқҙ н•„мҡ”н•ҳлӢӨ. н•ҳм§Җл§Ң мқҙлҘј кө¬н•ҳлҠ” кІғмқҖ м–ҙл ӨмӣҢм„ң CycleGANмқ„ мқҙмҡ©н•ҳм—¬ Image Restorationмқ„ м§„н–үн•ҳлҠ” кІҪмҡ°к°Җ мһҲлӢӨ. н•ҳм§Җл§Ң CycleGANкіј к°ҷмқҖ Image to Image translationкіј DIRTмҷҖ к°ҷмқҖ unsupervised domain adaptationмқҖ лӢӨмқҢкіј к°ҷмқҖ лӢЁм җмқ„ к°Җм§Җкі  мһҲлӢӨ.

Indistint Domain Boundary

Horse to zebraмҷҖ к°ҷмқҙ image translationмқҖ л¶„лӘ…н•ң domain boundaryк°Җ мЎҙмһ¬н•ңлӢӨ. н•ҳм§Җл§Ң image restorationмқҖ noise levelкіј ліөмһЎн•ң backgroundк°Җ domain boundaryлҘј нқ¬лҜён•ҳкІҢ л§Ңл“Өм–ҙм„ң мқҙлҜём§Җ нҖ„лҰ¬нӢ°к°Җ лӮ®м•„м§„лӢӨ.

Weak Representation

Unsupervised Domain AdaptationмқҖ high-level representationл§Ң м¶”м¶ңн•ңлӢӨ. мқҙлҠ” domain shift problemмқ„ м•јкё°н•ҳм—¬ low-quality reconstructionмқ„ л§Ңл“Өм–ҙлӮёлӢӨ.

Poor Generalization

One-to-one image translationмқҖ semantic representationкіј texture representationмқ„ л¶„лҰ¬н•ҳм—¬ мһЎм•„лӮҙкё° нһҳл“ӨлӢӨ.

л”°лқјм„ң мқҙ л…јл¬ём—җм„ң мң„мқҳ л¬ём ңлҘј н•ҙкІ°н•ҳл©° лӢӨмқҢмқҳ contributionмқ„ лӮЁкІјлӢӨ.

  1. Image restorationл¶„м•јм—җм„ң unsuperviesd representation learning methodлҘј м ңм•Ҳн–ҲлӢӨ.
  2. Dual domain constraintлҘј нҶөн•ҙ semantic representationкіј texture representation; л‘җ к°ңмқҳ representationмқ„ л¶„лҰ¬н–ҲлӢӨ.
  3. Domain transferлҘј нҶөн•ҙ unsupervised image restorationмқ„ м ңм•Ҳн–ҲлӢӨ.

The Proposed Method

лЁјм Җ л¬ём ңм—җ лҢҖн•ң м •мқҳлҘј н•ҳкІ лӢӨ. \(\mathcal{X}\)лҘј Noisy Image Domain, \(\mathcal{Y}\)лҘј Clean Image Domainмқҙлқјкі  н•ҳкІ лӢӨ. EncoderлҠ” к°Ғк°Ғмқҳ лҸ„л©”мқёмқ„ к°ҷмқҖ vector spaceмқё shared-latent space \(\mathcal{Z}\)лЎң projection мӢңнӮЁлӢӨ. л”°лқјм„ң vector spaceм—җ лҢҖн•ҳм—¬ лӢӨмқҢмқҳ мӢқмқҙ м„ұлҰҪн•ңлӢӨ.

\[z=E_\mathcal{X}(x)=E_\mathcal{Y}(y)\]

GeneratorлҠ” shared-latent space \(\mathcal{Z}\)м—җм„ң imageлҘј л§Ңл“Өм–ҙлӮёлӢӨ. л”°лқјм„ң лӢӨмқҢкіј к°ҷмқҖ мӢқмқҙ м„ұлҰҪн•ңлӢӨ.

\[x=G_\mathcal{X}(z),y=G_\mathcal{Y}(z)\]

мқҙ л•Ң к°Ғк°Ғмқҳ лҸ„л©”мқём—җлҢҖн•ҙ EncoderмҷҖ GeneratorлҠ” \(\{E_\mathcal{X}, G_\mathcal{X}\}, \{E_\mathcal{Y}, G_\mathcal{Y}\}\) к°Ғк°Ғ мЎҙмһ¬н•ңлӢӨ. к°Ғк°Ғмқҳ encoderк°Җ shared-latent space \(\mathcal{Z}\)лЎң projectionмқ„ мӢңнӮЁлӢӨкі  н•ҳлҚ”лқјлҸ„ к°Ғк°Ғмқҳ latent vectorлҠ” лӢӨлҘҙлӢӨ. л”°лқјм„ң latent vectorлҘј кө¬л¶„н•ҳм—¬ м Ғм–ҙмЈјкІ лӢӨ.

\[z_\mathcal{X}=E_\mathcal{X}(x), z_\mathcal{Y}=E_\mathcal{Y}(y)\]

л”°лқјм„ң мҡ°лҰ¬к°Җ н•ҳкі мӢ¶м–ҙн•ҳлҠ” Image restorationкіјм •мқҖ лӢӨмқҢкіј к°ҷлӢӨ.

\[F^{\mathcal{X}\rightarrow\mathcal{Y}}(x)=G_\mathcal{Y}(z_\mathcal{X})\]

Discrete Representation Learning

лЁјм Җ one-to-one image translationмқҳ poor generalization л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ semantic representationкіј texture(noise) representationмқ„ л¶„лҰ¬мӢңмјңм•јн•ңлӢӨ. л”°лқјм„ң м ҖмһҗлҠ” лӢӨмқҢ 4к°Җм§Җмқҳ л°©лІ•лЎ мқ„ м ңмӢңн–ҲлӢӨ.

  1. Detangling Representation
  2. Forward Cross Translation
  3. Backward Cross Reconstruction
  4. Adversarial Domain Adaptation

Detangling Representation

лЁјм Җ extra noise encoder(\(E_\mathcal{X}^N\))мқ„ лҸ„мһ…мқ„ н•ңлӢӨ. \(E_\mathcal{X}^N\)мқҖ noiseлҘј лӮҳнғҖлӮҙлҠ” texture latent vectorлҘј лҪ‘м•„лӮҙлҠ” м—ӯн• лЎң мқҙлҘј лҸ„мһ…н•ҙм„ң semantic representationкіј texture representationмқ„ л¶„лҰ¬н–ҲлӢӨ. мқҙлҘј нҶөн•ҙ \(z_\mathcal{X}\)мҷҖ \(z_\mathcal{Y}\)лҠ” к°ҷмқҖ distributionмқ„ к°Җм§ҖкІҢ лҗңлӢӨ. л§Ңм•Ҫ noise imageлҘј self-reconstructionн•ҳл Өл©ҙ \(x=G_\mathcal{X}(z_\mathcal{X}, z_\mathcal{X}^N)\)мқ„ нҶөн•ҳм—¬ к°ҷмқҙ reconstructionн•ҳл©ҙ лҗңлӢӨ.

Forward Cross Translation

CycleGANмІҳлҹј noise imageм—җм„ң clean image ліҖнҷҳкіј clean imageм—җм„ң noiseмқҙлҜём§Җмқҳ ліҖнҷҳмқҙ лҗҳм–ҙм•јн•ңлӢӨ. л”°лқјм„ң лӢӨмқҢкіј к°ҷмқҖ л°©лІ•мңјлЎң мқҙлҜём§Җ ліҖнҷҳмқ„ н•ңлӢӨ. мқҙ л•Ң \(\mathcal{Y}\)м—җ \(\mathcal{X}\)мқҳ noiseлҘј м¶”к°Җн•ҳкё° мң„н•ҙ \(z_\mathcal{X}^N\)лҘј мқҙмҡ©н•ңлӢӨ.

  1. Noise image вҶ’ clean image: \(\tilde{x}^{\mathcal{X}\rightarrow\mathcal{Y}} =G_\mathcal{Y}(z_\mathcal{X})\)
  2. Clean image вҶ’ noise image: \(\tilde{y}^{\mathcal{Y}\rightarrow\mathcal{X}} =G_\mathcal{X}(z_\mathcal{Y}\oplus z_\mathcal{X}^N)\)

Backward Cross Translation

Forward cross translationмқ„ н–ҲмңјлӢҲ backward cross translationмқ„ н•  мҲҳ мһҲлӢӨ. мқҙ л•Ң \(\mathcal{X}\)м—җ \(\mathcal{Y}\)мқҳ noiseлҘј м¶”к°Җн•ҳкё° мң„н•ҙ \(E_\mathcal{X}^N(\tilde{y}^{\mathcal{Y}\rightarrow\mathcal{X}})\)лҘј мқҙмҡ©н•ңлӢӨ.

  1. Noise image вҶ’ clean image: \(\hat{x}=G_\mathcal{X}(E_\mathcal{Y}(\tilde{x}^{\mathcal{X}\rightarrow\mathcal{Y}})\oplus E_\mathcal{X}^N(\tilde{y}^{\mathcal{Y}\rightarrow\mathcal{X}}))\)
  2. Clean image вҶ’ noise image: \(\hat{y}=G_\mathcal{Y}(E_\mathcal{X}(\tilde{y}^{\mathcal{Y}\rightarrow\mathcal{X}}))\)

Backward cross translatioлҘј н•ҷмҠөн•ҳкё° мң„н•ҙ lossлҘј лӢӨмқҢкіј к°ҷмқҙ кө¬м„ұн•ңлӢӨ.

\[\mathcal{L}_\mathcal{X}^{CC}(G_\mathcal{X},G_\mathcal{Y},E_\mathcal{X},E_\mathcal{Y},E_\mathcal{X}^N)=\mathbb{E}_\mathcal{X}[||G_\mathcal{X}(E_\mathcal{Y}(\tilde{x}^{\mathcal{X}\rightarrow\mathcal{Y}})\oplus E_\mathcal{X}^N(\tilde{y}^{\mathcal{Y}\rightarrow\mathcal{X}}))-x||_1]\] \[\mathcal{L}_\mathcal{Y}^{CC}(G_\mathcal{X},G_\mathcal{Y},E_\mathcal{X},E_\mathcal{Y},E_\mathcal{X}^N)=\mathbb{E}_\mathcal{X}[||G_\mathcal{Y}(E_\mathcal{X}(\tilde{y}^{\mathcal{Y}\rightarrow\mathcal{X}}))-y||_1]\]

Adversarial Domain Adaptation

Semantic representation (\(z_\mathcal{X}, z_\mathcal{Y}\))мқҖ к°ҷмқҖ vector spaceлҘј мӮ¬мҡ©н•ҙм•јн•ңлӢӨ. л”°лқјм„ң мқҙлҘј к°•м ңн•ҳкё° мң„н•ҙм„ң reprenentation discriminator \(D_r\)лҘј мӮ¬мҡ©н•ңлӢӨ.

\[\mathcal{L}^\mathcal{R}_{adv}(E_\mathcal{X},E_\mathcal{Y},D_\mathcal{R})=\mathbb{E}_\mathcal{X}[\frac{1}{2}logD_\mathcal{R}(z_\mathcal{X}+\frac{1}{2}(1-logD_\mathcal{R}(z_\mathcal{X})))] + \mathbb{E}_\mathcal{Y}[\frac{1}{2}logD_\mathcal{R}(z_\mathcal{Y}+\frac{1}{2}(1-logD_\mathcal{R}(z_\mathcal{Y})))]\]

Self-Supervised Constraint

Background Consistency Loss

ліөмһЎн•ң backgroundмқҳ Indistint Domain Boundaryмқ„ н•ҙкІ°н•ҳкё° мң„н•ҙ backgound consistency lossлҘј м ңм•Ҳн•ңлӢӨ. Noiseк°Җ мһҲлҠ” мқҙлҜём§ҖмҷҖ к№ЁлҒ—н•ң мқҙлҜём§ҖлҠ” gaussian blurлҘј н•ҳл©ҙ structure м •ліҙл§Ң лӮЁкё° л•Ңл¬ём—җ backgroundмқҳ л№„көҗк°Җ к°ҖлҠҘн•ҳлӢӨ. л”°лқјм„ң м ҖмһҗлҠ” лӢӨмқҢкіј к°ҷмқҖ lossлҘј м¶”к°Җн•ңлӢӨ.

\[\mathcal{L}_{BC}=\sum_{\sigma=5,9,15} \lambda_\sigma||B_\sigma(\mathcal{X})-B_\sigma(\tilde{\mathcal{X}})||_1\]

Semantic Consistency Loss

perception lossм—җм„ң мҳҒк°җмқ„ л°ӣм•„ pretrained-backboneмқ„ нҶөн•ң semanticн•ң м •ліҙлҠ” noiseк°Җ мӨ„м–ҙл“Ө кІғмқҙлқјкі  кё°лҢҖк°Җ лҗңлӢӨ. л”°лқјм„ң VGG19м—җм„ң conv5-1 layerмҷҖ к°ҷмқҙ к№ҠмқҖ layerм—җм„ң feature mapмқ„ лҪ‘м•„ л№„көҗн•ҳм—¬ semantic representationsмқҳ consistencyлҘј мң м§Җн•ңлӢӨ.

\[\mathcal{L}_{SC}=||\phi_l(\mathcal{X}-\phi_l(\tilde{\mathcal{X}})||_2^2\]

Joint Optimizing

мўӢмқҖ м„ұлҠҘмқ„ мң„н•ҳм—¬ лӢӨлҘё м—¬лҹ¬к°Җм§ҖлҸ„ м¶”к°Җн–ҲлӢӨ.

Target Domain Adversarial Loss

Noise Domainкіј clean image domainм—җм„ң кІ°кіјл¬јмқ„ лҚ” мһҳл§Ңл“Өкё° мң„н•ҙ GAN lossлҘј м¶”к°Җн•ңлӢӨ.

\[\mathcal{L}_{adv}^\mathcal{X}=\mathbb{E}_{x\sim P_\mathcal{X}(x)}[logD_\mathcal{X}(x)]+\mathbb{E}_{y\sim P_\mathcal{Y}(y), x \sim P_\mathcal{X}(x)}[log(1-D_\mathcal{X}(G_\mathcal{X}(E_\mathcal{Y}(y), E_\mathcal{X}^N(x))))]\] \[\mathcal{L}_{adv}^\mathcal{Y}=\mathbb{E}_{y\sim P_\mathcal{Y}(y)}[logD_\mathcal{Y}(y)]+\mathbb{E}_{x \sim P_\mathcal{Y}(y)}[log(1-D_\mathcal{Y}(G_\mathcal{Y}(E_\mathcal{X}(x)))]\]

Self Reconstruction Loss

м•Ҳм •м Ғмқё н•ҷмҠө м§„н–үмқ„ мң„н•ҙм„ң self reconstruction lossлҸ„ м¶”к°Җн•ҳмҳҖлӢӨ.

\[\hat{x}=G_\mathcal{X}(E_\mathcal{X}(x)\oplus E_\mathcal{X}^N(x)), \hat{y}=G_\mathcal{Y}(E_\mathcal{Y}(y))\] \[\mathcal{L}^\mathcal{X}_{rec}=||\hat{x} - x||_1, \mathcal{L}^\mathcal{Y}_{rec}=||\hat{y} - y||_1\]

KL Divergence Loss

NoiseлҠ” ліҙнҶө normal distributionмқ„ л”°лҘёлӢӨ. л”°лқјм„ң мқҙ л…јл¬ём—җм„ңлҸ„ latent-vectorк°Җ normal distributionмқ„ л”°лҘҙлҸ„лЎқ KL divergence lossлҘј м¶”к°Җн–ҲлӢӨ.

\[p(z_\mathcal{X}^N\sim N(0, 1))\]

Total Loss

лӘЁл“  lossлҘј н•©м№ҳл©ҙ лӢӨмқҢкіј к°ҷлӢӨ.

\[\underset{E_\mathcal{X},E_\mathcal{X}^N,E_\mathcal{Y},G_\mathcal{X},G_\mathcal{Y}}{\operatorname{min}} \underset{D_\mathcal{X},D_\mathcal{Y},D_\mathcal{R}}{\operatorname{max}} =\lambda_\mathcal{R}\mathcal{L}^\mathcal{R}_{adv}+\lambda_{adv}\mathcal{L}^{domain}_{adv}+\lambda_{CC}\mathcal{L}^{CC}+\lambda_{rec}\mathcal{L}^{Rec}+\lambda_{bc}\mathcal{L}^{BC}+\lambda_{sc}\mathcal{L}^{SC}+\lambda_{KL}\mathcal{L}^{KL}\]

Restoration

н•ҷмҠөмқҙ лҒқлӮң нӣ„м—җ noiseк°Җ мһҲлҠ” мқҙлҜём§ҖлҘј ліөмӣҗн•ҳл Өл©ҙ cross encoder-generator \(\{ E_\mathcal{X}, E_\mathcal{Y}\}\)лҘј мӮ¬мҡ©н•ҳл©ҙ лҗңлӢӨ.

\[\tilde{x}^{\mathcal{X}\rightarrow \mathcal{Y}}=G_\mathcal{Y}(E_\mathcal{X}(x))\]

Experiment

Unsupervised л°©лІ•л“Ө мӨ‘м—җм„ң SOTA м„ұлҠҘмқ„ лӢ¬м„ұн–ҲлӢӨ. н•өмӢ¬ кҙҖм°°:

  • Supervised л°©лІ• лҢҖл№„: лӢ№м—°нһҲ supervised л°©лІ•(pair лҚ°мқҙн„° мӮ¬мҡ©)ліҙлӢӨлҠ” м„ұлҠҘмқҙ лӮ®лӢӨ. н•ҳм§Җл§Ң pair лҚ°мқҙн„°к°Җ м—ҶлҠ” нҳ„мӢӨм Ғмқё мғҒнҷ©мқ„ кі л Өн•ҳл©ҙ мқҳлҜёмһҲлҠ” кІ°кіјмқҙлӢӨ.
  • CycleGAN лҢҖл№„: CycleGAN кё°л°ҳ л°©лІ•л“ӨмқҖ domain boundaryк°Җ л¶ҲлӘ…нҷ•н•ң image restorationм—җм„ң artifactмқҙ л°ңмғқн•ҳкё° мү¬мҡҙлҚ°, мқҙ л…јл¬ёмқҖ semantic/texture representation л¶„лҰ¬мҷҖ background consistency lossлЎң мқҙлҘј мҷ„нҷ”н–ҲлӢӨ.
  • м§Ҳм Ғ л№„көҗ: мӢңк°Ғм ҒмңјлЎң ліҙл©ҙ, кё°мЎҙ unsupervised л°©лІ•л“ӨмқҖ noiseлҘј м ңкұ°н•ҳл©ҙм„ң texture detailлҸ„ н•Ёк»ҳ мӮ¬лқјм§ҖлҠ” кІҪмҡ°к°Җ л§ҺмқҖлҚ°, мқҙ л…јл¬ёмқҳ л°©лІ•мқҖ noiseлҠ” м ңкұ°н•ҳл©ҙм„ң semantic structureлҘј мһҳ ліҙмЎҙн•ңлӢӨ. мқҙлҠ” dual domain constraintк°Җ л‘җ representationмқ„ нҡЁкіјм ҒмңјлЎң л¶„лҰ¬н–Ҳкё° л•Ңл¬ёмқҙлӢӨ.

мқҙ л…јл¬ёмқҳ мқҳмқҳ

Unsupervised image restorationмқҙлқјлҠ” м–ҙл Өмҡҙ л¬ём ңм—җ лҢҖн•ҙ, CycleGANмқҳ н•ңкі„(л¶ҲлӘ…нҷ•н•ң domain boundary, weak representation, poor generalization)лҘј мІҙкі„м ҒмңјлЎң л¶„м„қн•ҳкі , к°Ғ л¬ём ңм—җ лҢҖмқ‘н•ҳлҠ” solution(background consistency loss, adversarial domain adaptation, detangling representation)мқ„ м ңм•Ҳн•ң кІғмқҙ н•өмӢ¬ кё°м—¬мқҙлӢӨ. лӢӨл§Ң lossк°Җ 7к°ңлЎң л§Өмҡ° л§Һм•„м„ң hyperparameter tuningмқҙ м–ҙл Өмҡё мҲҳ мһҲлӢӨлҠ” м җмқҖ мӢӨмҡ©м Ғ н•ңкі„мқҙлӢӨ.
