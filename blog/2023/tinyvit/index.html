TinyViT: Fast Pretraining Distillation for Small Vision Transformers (Wu et al., Microsoft, ECCV 2022)

Introduction

ViTлҠ” л§ҺмқҖ л°ңм „мқҙ мһҲм—Ҳм§Җл§Ң edge deviceм—җ м Ғмҡ©н•ҳкё°м—җлҠ” лӘЁлҚёмқҙ л„Ҳл¬ҙ м»ёлӢӨ. н•ҳм§Җл§Ң лӘЁл‘җл“Ө м•ҢлӢӨмӢ¶мқҙ мһ‘мқҖ лӘЁлҚё representationмқҙ мһ‘лӢӨ. л”°лқјм„ң small datasetм—җ м Ғн•©н• м§Җ лӘ°лқјлҸ„ large datasetм—җм„ңлҠ” л№ лҘҙкІҢ saturationмқҙ лҗҳл©ҙм„ң underfittingмқҙ л°ңмғқн•ҳкІҢ лҗҳм–ҙ мһҳ мӮ¬мҡ©н•ҳм§Җ лӘ»н•ңлӢӨ. м ҖмһҗлҠ” мқҙм—җ лҢҖн•ҳм—¬ кі лҜјмқ„ н–Ҳкі  knowledge distillationмқ„ мӮ¬мҡ©н•ҙм•јн•ңлӢӨлҠ” кІ°лЎ м—җ лҸ„лӢ¬н•ҳкІҢ лҗңлӢӨ. мқҙм—җл”°лқј small modelмқҙ downstream taskм—җ transferк°Җ мһҳ лҗҳлҠ” кІғмқ„ нҷ•мқён–ҲлӢӨ. н•ҳм§Җл§Ң кё°мЎҙмқҳ knowledge distillationмқҖ trainмӢң teacher networkлҘј л©”лӘЁлҰ¬м—җ мҳ¬лҰ¬кё° л•Ңл¬ём—җ teacher networkк°Җ gpu memoryлҘј лӢӨмҲҳ мһЎм•„лЁ№кІҢ лҗҳм–ҙ batch size мЎ°м Ҳмқҙ нһҳл“ӨлӢӨ. лҳҗн•ң н•ҷмҠөм—җ н•„мҡ”н•ң soft-labelмқ„ к·ёл•Ңк·ёл•Ң м ңмһ‘н•ҳкё° л•Ңл¬ём—җ н•ҷмҠөмҶҚлҸ„лҸ„ лҠҗл Өм§„лӢӨ. л”°лқјм„ң м ҖмһҗлҠ” мқҙлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ data augmentationкіј soft-labelмқ„ лЁјм Җ м ҖмһҘн•ҳм—¬ student modelмқҙ н•ҷмҠө мӢң мқҙлҘј мӮ¬мҡ©н•ҳлҠ” л°©лІ•мқ„ мӮ¬мҡ©н–ҲлӢӨ. кІ°лЎ м ҒмңјлЎң TinyViTлҠ” 21Mмқҳ нҢҢлқјлҜён„°лЎң ImageNetм—җм„ң 84.8% top-1 accuracyлҘј лӢ¬м„ұн–Ҳмңјл©° 88MмңјлЎң 85.8%лҘј лӢ¬м„ұн•ң Swin-BліҙлӢӨ 4.2л°° м ҒмқҖ нҢҢлқјлҜён„°мқҙлӢӨ. мқҙлҜём§Җ нҒ¬кё°лҘј нӮӨмӣ мқ„ л•Ң SOTAлҘј м°Қм—Ҳмңјл©° COCO object detectionлҸ„ Swin-T мҡ°мң„м—җ мһҲлӢӨлҠ” кІғмқ„ нҷ•мқён–ҲлӢӨ.

TinyViT

Fast Pretraining Distillation

мң„ мӮ¬м§„м—җм„ң ліј мҲҳ мһҲл“Ҝ small modelмқ„ л°”лЎң нҒ° лҚ°мқҙн„°м…Ӣм—җ н•ҷмҠөн•ҳл©ҙ м„ұлҠҘмқҙ лӮ®м•„м§„лӢӨ. л”°лқјм„ң knowledge distillationмқ„ мқҙмҡ©н•ҳл Өкі  н•ҳлҠ”лҚ° м ҖмһҗлҠ” downstream taskлҘј мң„н•ҙ finetuning-distillationмқҙ м•„лӢҲлқј pretraining distillationм—җ мЈјлӘ©мқ„ н•ңлӢӨ. мқҙм—җ л”°лқј ImageNet-1Kм—җлҠ” distillationм—Ҷмқҙ finetuningмқ„ мқҙмҡ©н•ҳм—¬ н•ҷмҠөмқ„ м§„н–үн•ңлӢӨ. н•ҳм§Җл§Ң pretraining distillationмқҖ large teacher modelмқҙ л§ҺмқҖ лҚ°мқҙн„°м…Ӣмқ„ inference н•ҙм•ј н•ҳкё° л•Ңл¬ём—җ л№„мҡ©мқҙ л§Һмқҙ л“Өкі  л№„нҡЁмңЁм ҒмқҙлӢӨ. л”°лқјм„ң Fig.2м—җм„ң лӮҳмҳӨл“Ҝ мӮ¬м „м—җ teacher modelм—җ мӮ¬мҡ©лҗҳлҠ” augmentationкіј к·ём—җ л§һлҠ” labelмқ„ л§Ңл“Өм–ҙ мҠӨнҶ лҰ¬м§Җм—җ м ҖмһҘн•ҳкІҢ лҗңлӢӨ. мқҙлҘј нҶөн•ҙ student modelмқҙ н•ҷмҠө мӢң teacher modelмқҳ forward computation м—Ҷмқҙ knoweldge distillationмқ„ н•  мҲҳ мһҲлӢӨ. мҲҳн•ҷм ҒмңјлЎң н‘ңнҳ„н•ҳмһҗл©ҙ, input imageлҘј \(x\), RandAugmentлӮҳ CutMixмҷҖ к°ҷмқҖ strong augmentationмқ„ \(\mathcal{A}\)лқј н•ҳмһҗ. Teacher н•ҷмҠөмӢң augmentation \(\mathcal{A}\), techer prediction \(\hat{y}=T(\mathcal{A}(s))\)мқҳ pairмқё \((\mathcal{A},\hat{y})\)лҘј м ҖмһҘн•ңлӢӨ. мқҙ л•Ң \(T(\cdot )\)мқҖ teacher modelмқ„ мқҳлҜён•ңлӢӨ. к·ё нӣ„ student \(S(\cdot)\)кіј cross entropy \(CE(\cdot)\)м—җ лҢҖн•ҙ loss м—°мӮ°мқ„ м§„н–үн•ңлӢӨ.

\[\mathcal{L}=CE(\hat{y},S(\mathcal{A}(x)))\]

Sparse Soft Label

Teacher modelмқҳ outputмқ„ к·ёлҢҖлЎң м ҖмһҘн•ҳлҠ” кІғмқҖ storageлҘј л§Һмқҙ мӮ¬мҡ©н•ңлӢӨ. л”°лқјм„ң м ҖмһҗлҠ” \(\hat{y}\)мқҳ top-K valueкіј к·ёл“Өмқҳ indices \(\{\mathcal{I}(k)\}_{k=1}^K\)л§Ңмқ„ м ҖмһҘн•ҳкі  лӮҳлЁём§ҖлҠ” label smoothingмқ„ мқҙмҡ©н•ҳм—¬ reconstructн•ңлӢӨ. н•ҷмҠө мӢң ground truthмқё hard labelмқ„ мӮ¬мҡ©н•ҳм§Җ м•Ҡкі  pseudo labelмқё soft labelл§Ңмқ„ мӮ¬мҡ©н•ҳм—¬ н•ҷмҠөн•ңлӢӨ.

\[\hat{y}_c = \begin{cases} \hat{y}_{\mathcal{I}(k)} & \text{if} \ c=\mathcal{I}(k)\\ \frac{1-\sum_{k=1}^K\hat{y}_{\mathcal{I}(k)}}{C-K} & \text{otherwise} \end{cases}\]

мқҙ л•Ң \(\hat{y}_c\)лҠ”recovered teacher logit for student model distillationмқҙлқјкі  н•ҳкі , \(\hat{y}=[\hat{y}_1, ... ,\hat{y}_c, ... ,\hat{y}C]\)мқҙлӢӨ. л§Ңм•Ҫ \(K << C\)лқјл©ҙ л©”лӘЁлҰ¬ к°җмҶҢк°Җ нҒ¬лӢӨ.

Data augmentation encoding

Data augmentation м •ліҙ лҳҗн•ң к·ёлҢҖлЎң м ҖмһҘн•ҳл©ҙ storageлҘј л§Һмқҙ мӮ¬мҡ©н•ңлӢӨ. (rotation degree, crop coordinate л“ұ) л”°лқјм„ң set of data augmenatation parameterлҘј \(\mathbf{d}\)лқјкі  н•ҳкі , encoder \(\mathcal{E}(\cdot)\) лҘј мӮ¬мҡ©н•ҳм—¬ \(d_0=\mathcal{E}(\mathbf{d})\)лЎң ліҖнҷҳн•ҳм—¬ м ҖмһҘн•ңлӢӨ. к·ё нӣ„ training processм—җлҠ” \(\mathcal{E}^{-1}(\cdot)\)мқ„ decoderлЎңмӮ¬мҡ©н•ҳм—¬ augmetationмқ„ м§„н–үн•ңлӢӨ.

Model Architecture

м ҖмһҗлҠ” кё°ліём ҒмңјлЎң ViTлҘј кё°л°ҳмңјлЎң лӘЁлҚёмқ„ м„Өкі„н–ҲлӢӨ. лҳҗн•ң Swin TransformerлӮҳ LeViTмҷҖ к°ҷмқҙ hierarchicalн•ң кө¬мЎ°лҘј мұ„нғқн–ҲлӢӨ. Patch embedding blockмқҖ kernel size 3, stride 2, padding 1мқё л‘җ к°ңмқҳ convolutionмқ„ мӮ¬мҡ©н–ҲлӢӨ. н•ҳм§Җл§Ң мІҳмқҢл¶Җн„° лҒқк№Ңм§Җ transformer blockмқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҖ м—°мӮ°м ҒмңјлЎң л¬ҙлҰ¬к°Җ лҗңлӢӨ. л”°лқјм„ң MobileNetV2м—җм„ң мӮ¬мҡ©н•ҳлҠ” MBConvлҘј мӮ¬мҡ©н•ҳм—¬ stage 1кіј downsamplingмқ„ кө¬м„ұн–ҲлӢӨ. лҳҗн•ң MBConvм—җ residual connectionлҸ„ м Ғмҡ©н–ҲлӢӨ. л§Ҳм§Җл§ү 3к°ң stage м—җм„ңлҠ” transformer blockмқ„ мӮ¬мҡ©н–ҲлӢӨ. лӘЁл“  layerмҷҖ blockм—җм„ң activation functionмқҖ мҡ”мҰҳ м„ұлҠҘмқҙ мўӢмқҖ GELUлҘј мқҙмҡ©н–Ҳкі , normalizationмқҖ conolution layerм—җлҠ” batch norm, linear layerм—җлҠ” layer normмқ„ мӮ¬мҡ©н–ҲлӢӨ.

Contraction factors

Modern modelмҷҖ к°ҷмқҙ factorлҘј мӮ¬мҡ©н•ҳм—¬ лӘЁлҚёмқҳ нҒ¬кё°лҘј мЎ°м Ҳн•ңлӢӨ.

  • \(\gamma_{D_{1-4}}\): embeded demension in four stages
  • \(\gamma_{N_{1-4}}\): number of block in four stages
  • \(\gamma_{W_{2-4}}\): window size of last three stages
  • \(\gamma_{R}\): channel expansion ratio of MBConv
  • \(\gamma_{M}\): channel expansion MLP of transformer blocks
  • \(\gamma_{E}\): the dimension of each head in multi-head attention

лӘЁл“  лӘЁлҚёмқҙ \(\gamma_{D_{1-4}}\)мқ„ м ңмҷён•ҳкі  к°ҷмқҖ factorлҘј к°–лҠ”лӢӨ.

\[\{\gamma_{N_1}, \gamma_{N_1}, \gamma_{N_1}, \gamma_{N_1}\} = \{2, 2, 6, 2\}\] \[\{\gamma_{W_1}, \gamma_{W_1}, \gamma_{W_1}\} = \{7, 14, 7\}\] \[\{\gamma_{R}, \gamma_{M}, \gamma_{E}\} = \{4, 4, 32\}\]

Embeded dimensiondm \(\{\gamma_{D_1}, \gamma_{D_2}, \gamma_{D_3}, \gamma_{D_4}\}\)лҠ” TinyViT-21M: {96, 192, 384, 576},TinyViT-11M: {64, 128, 256, 448},TinyViT-5M: {64, 128, 160, 320} мңјлЎң кө¬м„ұн–ҲлӢӨ.

Analysis and Discussion

мһ‘мқҖ лӘЁлҚёмқҖ Image21KмҷҖ к°ҷмқҖ large scale datasetм—җм„ңлҠ” underfittingмқҙ л°ңмғқн•ҳм—¬ н•ҷмҠөмқҙ мһҳлҗҳм§Җ м•ҠлҠ”лӢӨ. к·ёл ҮлӢӨл©ҙ лӢӨмқҢ л‘җ к°Җм§Җмқҳ мқҳл¬ёмқҙ л“Ө мҲҳ мһҲлӢӨ.

1. м–ҙл– н•ң мҡ”мҶҢк°Җ small modelмқҳ н•ҷмҠөмқ„ л°©н•ҙн•ҳлҠ”к°Җ?

мқҙлҘј нҷ•мқён•ҳл Өл©ҙ IN-21Kмқҳ л¬ём ңм җмқ„ м•Ңм•„м•јн•ңлӢӨ. Labelмқҳ мҳӨлҘҳмҷҖ л№„мҠ·н•ң мқҙлҜём§Җк°Җ лӢӨлҘё labelмқё кІҪмҡ°к°Җ мЎҙмһ¬н•ңлӢӨ. мқҙмҷҖ к°ҷмқҖ лҚ°мқҙн„°к°Җ м „мІҙмқҳ 10%лЎң hard sampleмқҙлқјкі  л¶ҲлҰ°лӢӨ. Large modelмқҖ мқҙлҹ¬н•ң hard sampleмқ„ н•ҷмҠөн•  мҲҳ мһҲмһҗл§Ң small modelмқҖ hard sampleмқ„ мҳҲмёЎн•ҳкё° м–ҙл өкі  мқҙлҠ” small modelмқ„ large model ліҙлӢӨ лӮ®мқҖ м„ұлҠҘмңјлЎң мқҙлҒҲлӢӨ. мқҙлҘј мҰқлӘ…н•ҳкё° мң„н•ҙ к°•л Ҙн•ң лӘЁлҚёмқё FlorenceлҘј мқҙмҡ©н•ҳм—¬ dataл§ҲлӢӨ florence predictionмқҳ top 5 мӨ‘м—җ labelмқҙ мЎҙмһ¬н•ҳлҠ”м§Җ м°ҫм•„лҙӨлӢӨ. к·ё кІ°кіј ImageNet-21Kмқҳ 14%м—җ н•ҙлӢ№н•ҳлҠ” hard sampleмқ„ кіЁлқјлғҲлӢӨ. лҳҗн•ң FlorenceлҘј мқҙмҡ©н•ҳм—¬ TinyViT-21M/Swin-Tм—җ knowledge distllationмқ„ м Ғмҡ©н–ҲлӢӨ. к·ё кІ°кіј small modelмқ„ кі§л°”лЎң ImageNet-21Kм—җ н•ҷмҠөн•ҳлҠ” кІғмқҖ м„ұлҠҘм—җм„ң м ңм•Ҫмқҙ мЎҙмһ¬н–Ҳкі , ImageNet-21Kм—җм„ң hard sampleмқ„ м ңкұ°н•ҳл©ҙ 1%p м •лҸ„ м„ұлҠҘн–ҘмғҒмқҙ мһҲм—ҲлӢӨ. лҶҖлқјмҡҙ м җмқҖ knowledge distillationмқҙ hard sampleмқҳ defactмқ„ мӨ„м—¬мӨ¬лӢӨ.

2. мҷң distillationмқҙ small modelмқҳ large data н•ҷмҠөм—җ лҸ„мӣҖмқҙ лҗҳлҠ”к°Җ?

Student modelмқҙ teacher modelмқҳ м§ҖмӢқмқ„ л°”лЎң н•ҷмҠөн•  мҲҳ мһҲкё° л•Ңл¬ёмқҙлӢӨ. Gound truthлҠ” к°Ғ л¬јмІҙк°„ мғҒк°„кҙҖкі„лҘј ліҙм—¬мЈјм§Җ лӘ»н•ңлӢӨ. н•ҳм§Җл§Ң teacher modelмқҳ inference к°’мқҖ к·ёкІғмқ„ м•Ң мҲҳ мһҲлӢӨ.

мң„мқҳ к·ёлҰјмқ„ ліҙм•ҳмқ„ л•ҢлҸ„ distillation м—Ҷмқҙ TinyViTлҘј н•ҷмҠөн–Ҳмқ„ л•Ң л№„мҠ·н•ң л¬јмІҙк°„ мғҒкҙҖкҙҖкі„к°Җ нҒ¬м§Җ м•Ҡм•ҳмңјлӮҳ distillation м§„н–үмӢң мғҒкҙҖкҙҖкі„к°Җ нҒ¬кІҢ лӮҳмҷҖ teacher modelмқ„ м ңлҢҖлЎң л”°лқјн•  мҲҳ мһҲм—ҲлӢӨ.
