Meta Pseudo Labels

Introduction

Semi-supervised learning л°©лІ•мқҖ м—¬лҹ¬к°Җм§Җ мһҲлҠ”лҚ° к·ё мӨ‘м—җ н•ң к°Җм§ҖлҠ” pseudo labeling л°©лІ•мқҙлӢӨ. Pseudo labelingмқҖ мһҳ н•ҷмҠөлҗң teacher networkмҷҖ student networkк°Җ мЎҙмһ¬н•ҳлҠ”лҚ° teacher modelмқҖ unlabeled dataмқҳ pseudo labelмқ„ м ңмһ‘н•ҳкі  studentлҠ” к·ё labelмқ„ н•ҷмҠөн•ҳлҠ” кІғмңјлЎң м§„н–үлҗңлӢӨ.

н•ҳм§Җл§Ң мқҙ л°©лІ•мқҳ л¬ём ңм җмқҖ teacher modelмқҳ pseudo labelмқҙ м •нҷ•н•ҳм§Җ м•ҠлӢӨл©ҙ student modelмқҖ teacher modelмқҳ нҷ•мҰқнҺён–ҘмңјлЎң мқён•ҳм—¬ мһҳлӘ»лҗң л°©н–ҘмңјлЎң н•ҷмҠөмқ„ м§„н–үн•ңлӢӨ. л”°лқјм„ң м ҖмһҗлҠ” мқҙлҹ¬н•ң л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ teacher modelмқҙ labeled dataм—җ лҢҖн•ң student modelмқҳ м„ұлҠҘмқ„ нҷ•мқён•ҳл©ҙм„ң biasлҘј мҲҳм •н•ҳлҠ” кіјм •мқ„ м ңм•Ҳн•ңлӢӨ.

Meta Pseudo Label

л“Өм–ҙк°Җкё°м—җ м•һм„ң notationм—җ лҢҖн•ҳм—¬ м„ӨлӘ…н•ҳкІ лӢӨ.

  • \(T(x_u,\theta_T)\): Soft prediction of teacher model on unlabeld data
  • \(S(x_u,\theta_S)\): Soft prediction of student model on unlabeld data
  • \(S(x_l,\theta_S)\): Soft prediction of student model on labeled data
  • \(CE(q,p)\): cross-entropy

Meta pseudo labelмқҖ лӢӨмқҢкіј к°ҷмқҖ кіјм •мқ„ кұ°м№ңлӢӨ.

  1. Teacher modelмқҙ unlabeld dataмқҳ pseudo labelмқ„ мғқм„ұн•ңлӢӨ.
  2. Student modelмқҙ мқҙлҘј н•ҷмҠөн•ңлӢӨ.
  3. Studentмқҙ labeled dataлҘј мқҙмҡ©н•ҳм—¬ м„ұлҠҘмқ„ нҸүк°Җн•ңлӢӨ.
  4. Teacher modelмқҖ 3лІҲмқ„ мқҙмҡ©н•ҳм—¬ лҚ” мўӢмқҖ pseudo labelмқ„ л§Ңл“ӨлҸ„лЎқ н•ҷмҠөн•ңлӢӨ.

лҚ” мқҙн•ҙн•ҳкё°мүҪкІҢ мӢқмңјлЎң мӮҙнҺҙліҙмһҗ.

Studentмқҳ optimal parameterлҠ” teacher modelмқҙ м ңмһ‘н•ң pseudo-labelкіј student modelмқҳ мҳҲмёЎк°’мқҳ cross entropyк°Җ лӮ®мқҖ parameterмқҙлӢӨ.

\[\theta_S^{PL}=\underset{\theta_S}{\operatorname{argmin}} \mathbb{E}[CE(T(x_u;\theta_T), S(x_u;\theta_S)]\]

un-labeled datalossлҠ” лӢӨмқҢкіј к°ҷмқҙ м •мқҳн•  мҲҳ мһҲлӢӨ.

\[\mathbb{E}[CE(T(x_u;\theta_T), S(x_u;\theta_S)] := \mathcal{L}_u(\theta_T, \theta_S)\]

к·ёлҰ¬кі  teacher лӘЁлҚёмқҙ мӮ¬мҡ©н•ҳлҠ” labeled data lossлҠ” лӢӨмқҢкіј к°ҷмқҙ м •мқҳн•  мҲҳ мһҲлӢӨ.

\[\mathbb{E}_{x_l,y_l}[CE(y_l,S(x_l;\theta_S^{PL}))] := \mathcal{L}_l(\theta_S^{PL})\]

мқҙ л•Ң student modelмқҖ teacher modelмқҳ pseudo-labelмқ„ мқҙмҡ©н•ҳм—¬ н•ҷмҠөн•ҳкё° л•Ңл¬ём—җ м„ұлҠҘмқҖ teacher modelм—җ мқҳмЎҙм Ғмқё кІғмқ„ м•Ң мҲҳ мһҲлӢӨ. мқҙлҹ¬н•ң мқҙмң лЎң meta pseudo labelмқҙлқјкі  лӘ…лӘ…н•ҳмҳҖкі  мқҙлҘј лӮҳнғҖлӮҙкё° мң„н•ҙм„ң Notationмқ„ лӢӨмқҢкіј к°ҷмқҙ мһ‘м„ұн•ңлӢӨ.

\[\theta_S^{PL} \rightarrow \theta_S^{PL}(\theta_T)\]

лҳҗн•ң label data lossлҠ” лӢӨмқҢкіј к°ҷмқҙ лӮҳнғҖлӮј мҲҳ мһҲлӢӨ.

\[\mathcal{L}_l(\theta_S^{PL}) \rightarrow \mathcal{L}_l(\theta_S^{PL}(\theta_T))\]

techer modelмқҖ нҷ•мҰқнҺён–Ҙмқ„ мҲҳм •н•ҳкё° мң„н•ҙ \(\mathcal{L}_l(\theta_S^{PL}(\theta_T))\)мқ„ мқҙмҡ©н•ҳм—¬ parameterлҘј updateн•ңлӢӨ. н•ҳм§Җл§Ң мқҙлҘј м§Ғм ‘ кі„мӮ°н•ҳлҠ” кІғмқҖ нһҳл“Өкё° л•Ңл¬ём—җ approximationмқ„ н•ңлӢӨ.

\[\theta_S^{PL}(\theta_T) \approx \theta_S - \eta_S \cdot \triangledown_{\theta_S}\mathcal{L}_u(\theta_{T}, \theta_{S})\]

л”°лқјм„ң teacher modelмқҖ лӢӨмқҢмқ„ мөңмҶҢнҷ”н•ҳлҠ” кІғмқ„ лӘ©н‘ңлЎңн•ңлӢӨ.

\[\underset{\theta_T}{\operatorname{min}} \mathcal{L}_l (\theta_S - \eta_S \cdot \triangledown_{\theta_S}\mathcal{L}_u(\theta_{T}, \theta_{S}))\]

м •лҰ¬

Student modelмқҖ SGDлЎң pseudo-labelм—җ лҢҖн•ҳм—¬ optimizeн•ңлӢӨ.

\[\theta^{\prime}_S = \theta_S - \eta_S \triangledown_{\theta_S} \mathcal{L}(\theta_T, \theta_S)\]

Teacher modelмқҖ studentмқҳ gradient updateлҘј мһ¬мӮ¬мҡ©н•ҳм—¬ SGDлЎң optimizeн•ңлӢӨ.

\[\theta^{\prime}_T=\theta_T - \eta_T \triangledown_{\theta_T} \mathcal{L}_l (\theta_S - \eta_S \cdot \triangledown_{\theta_S}\mathcal{L}_u(\theta_{T}, \theta_{S}))\]

Auxiliary Loss

мқҙмҷём—җлҸ„ lossлҘј м¶”к°Җн•ҳл©ҙ м„ұлҠҘн–ҘмғҒм—җ кё°м—¬н•ңлӢӨ. л”°лқјм„ң teacher modelмқҖ л‘җ к°Җм§Җ lossлҘј м¶”к°Җн–ҲлӢӨ.

  1. Supervised: train on label
  2. Semi-supervised: UDA objective on unlabeld data

StudentлҠ” мҳӨм§Ғ Meta Pseudo labelлЎңл§Ң н•ҷмҠөмқ„ м§„н–үн•ҳмҳҖлӢӨ. мқҙнӣ„м—җлҠ” taskм—җ л§һлҸ„лЎқ finetuningмқ„ н•ҳмҳҖлӢӨ.

Experiment

Two moon dataset

к°Ғк°Ғмқҳ classл§ҲлӢӨ unlabled data 1000к°ңм”©, label data 3к°ңм”© м¶”м¶ңн•ҳмҳҖлӢӨ. Supervised л°©лІ•мқҖ label dataлҘј мһҳ л¶„лҘҳн•ҳм§Җл§Ң лӢӨлҘё лҚ°мқҙн„°лҠ” мҳӨлҘҳлҘј ліҙмқҙкі  мһҲлӢӨ. Supervised л°©лІ•мңјлЎң н•ҷмҠө лҗң teacher modelмқ„ нҶөн•ҳм—¬ pseudo labelмқ„ л§Ңл“Өм—Ҳмқ„л•ҢлҠ” label data мЎ°м°Ё м •нҷ•н•ҳкІҢ нҢҗлі„мқ„ лӘ»н–ҲлӢӨ. н•ҳм§Җл§Ң Meta Pseudo Labelм—җм„ңлҠ” м •нҷ•н•ҳкІҢ л‘җ classлҘј л¶„лҰ¬н•ҳмҳҖлӢӨ.

Small Model

EfficientNetкіј к°ҷмқҖ large modelмқ„ мӢӨн—ҳн•ҳкё° м „м—җ small modelлЎң мӢӨн—ҳмқ„ м§„н–үн•ҳмҳҖлӢӨ. м„ё к°Җм§Җ datasetмқ„ мӮ¬мҡ©н•ҳмҳҖлҠ”лҚ° CIFAR-10-4K, SVHNмқҖ WideResNet28-2лҘј мӮ¬мҡ©н•ҳмҳҖкі  imagenetмқҖ resnet50мқ„ мӮ¬мҡ©н•ҳмҳҖлӢӨ.

лӢӨлҘё SOTA methodл“Ө ліҙлӢӨ лҚ” мўӢмқҖ м„ұлҠҘмқ„ ліҙмҳҖлӢӨ.
